# Store Assets Checklist

Use this checklist to prepare store-ready assets for iOS and Android releases.

## Required Assets

| Asset | Platform | Specs | Status | Notes |
| --- | --- | --- | --- | --- |
| App Icon | iOS | 1024x1024 PNG, no transparency | [ ] | |
| App Icon | Android | 1024x1024 PNG + adaptive foreground | [ ] | |
| Splash Screen | iOS | 2732x2732, centered content | [ ] | |
| Splash Screen | Android | 2732x2732, centered content | [ ] | |
| Screenshots | iOS | 6.5" and 5.5" | [ ] | |
| Screenshots | Android | 7" and 10" | [ ] | |
| Feature Graphic | Android | 1024x500 PNG/JPG | [ ] | |

## Optional Assets

| Asset | Platform | Specs | Status | Notes |
| --- | --- | --- | --- | --- |
| Promo Video | iOS | 15-30 seconds | [ ] | |
| Promo Video | Android | 15-30 seconds | [ ] | |
| App Preview | iOS | 15-30 seconds | [ ] | |

## Validation Steps

- [ ] Ensure assets match current branding (colors, typography, logo).
- [ ] Confirm safe areas and text legibility on all sizes.
- [ ] Verify no test data appears in screenshots.
- [ ] Keep filenames consistent and versioned.
